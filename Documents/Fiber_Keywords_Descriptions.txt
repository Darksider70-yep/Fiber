
# 🌿 Fiber Language — Comprehensive Keyword List with Concepts and Descriptions

## 1. Core Language and Control

| Keyword | Concept | Description |
|----------|----------|-------------|
| var | Variable declaration | Declares a mutable variable |
| const | Constant declaration | Declares an immutable variable |
| static | Shared scope variable | Value shared across all instances or scopes |
| final | Assignment restriction | Value can only be assigned once |
| def | Function definition | Defines a reusable block of logic |
| class | Object definition | Defines a class or blueprint |
| struct | Lightweight data container | Defines a structured type with fields |
| enum | Enumeration | Defines symbolic constants |
| if / elif / else | Conditional flow | Standard branching logic |
| while | Loop | Executes while condition is true |
| for | Iteration | Standard or range-based loop |
| break | Loop control | Exits loop immediately |
| continue | Loop control | Skips to next iteration |
| return | Function exit | Returns a value from a function |
| import / from / as | Module management | Imports code from other files |
| try / catch / finally | Error handling | Handles and manages exceptions |
| throw | Raise exception | Manually triggers an error |
| assert | Runtime check | Verifies condition validity |
| pass | Placeholder | No-op, used for syntactic completeness |

## 5. State, Memory, and Scope Modifiers

| Keyword | Concept | Description |
|----------|----------|-------------|
| mutable | Variable mutability | Declares explicitly mutable variable |
| immutable | Variable immutability | Locks variable against change |
| volatile | Dynamic state | Marks variable as rapidly changing |
| lazy | Deferred evaluation | Evaluates when first accessed |
| cache | Cached computation | Saves results for reuse |
| shared | Shared memory | Accessible by concurrent scopes |
| sync | Synchronous execution | Ensures operation completes before proceeding |
| async | Asynchronous execution | Enables parallel or deferred task execution |
| yield | Iteration control | Returns intermediate values in generators |
| await | Async control | Waits for async result completion |
